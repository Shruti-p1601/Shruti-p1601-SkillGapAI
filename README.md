# SkillGapAI

SkillGapAI is a Python-based project that analyzes the skill gap between a resume and a job description.

## Project Structure
- milestone1.py – Resume data extraction
- milestone2.py – Job description processing
- milestone3.py – Skill matching logic
- milestone4.py – Skill gap analysis & output

## How to Run
1. Clone the repository
2. Navigate to SkillGapAI folder
3. Run any milestone file:
   python milestone1.py

## Tools Used
- Python
- NLP basics
- Git & GitHub
